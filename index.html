Problem Statement

Ranking Areas by Water Scarcity in Urban Areas

In urban areas, water scarcity is a growing issue, and efficient water management requires accurate identification and prioritization of regions most in need of intervention.

Why Is It Important?

  • Optimized Resource Allocation: Helps city planners and authorities allocate water resources to the areas most in need.
  • Equitable Distribution: Ensures that every area receives adequate water supply based on its scarcity level.
  • Cost-Effective Solutions: Enables targeted interventions, saving costs by focusing on the regions where water management issues are most critical.

What Happens When It Is Solved?

  • Improved Decision-Making: Decision-makers can make informed choices about where to implement water-saving measures, build infrastructure, or provide aid.
  • Reduced Wastage: Helps reduce the wastage of water by directing resources efficiently to areas where they are most needed.
  • Sustainable Urban Development: Contributes to sustainable urban growth by ensuring that water resources are managed in an equitable and efficient manner.

Algorithm: Merge Sort

What is Merge Sort?

Merge Sort is a divide-and-conquer sorting algorithm that breaks down a dataset into smaller sub-arrays, sorts them, and then merges them back together to form a sorted list.

Why Merge Sort?

  • Efficiency: O(n log n) time complexity for all cases
  • Stability: Maintains relative order of equal elements
  • Parallelizable: Suitable for parallel processing

Time Complexity

  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Worst Case: O(n log n)

Implementation


#include <iostream>
#include <vector>
#include <string>
using namespace std;

void merge(vector<pair<string, int>>& areas, int left, int mid, int right) {
    int n1 = mid - left + 1;
    int n2 = right - mid;
    
    vector<pair<string, int>> L(n1), R(n2);
    for (int i = 0; i < n1; i++)
        L[i] = areas[left + i];
    for (int j = 0; j < n2; j++)
        R[j] = areas[mid + 1 + j];
        
    int i = 0, j = 0, k = left;
    
    while (i < n1 && j < n2) {
        if (L[i].second <= R[j].second) {
            areas[k] = L[i];
            i++;
        } else {
            areas[k] = R[j];
            j++;
        }
        k++;
    }
    
    while (i < n1) {
        areas[k] = L[i];
        i++;
        k++;
    }
    
    while (j < n2) {
        areas[k] = R[j];
        j++;
        k++;
    }
}
